# rest-api-nodejs-template
Sample NodeJS REST API template with mysql2

## DB
To create the db simply use the db.sql which is in this repository.

## .env Structure
```
PORT = 3000
DB_HOST = localhost
DB_PORT = 3306
DB_USER = <your_user>
DB_PASS = <your_password>
DB = <your_db_name>

MY_SECRET_KEY = 42
```
